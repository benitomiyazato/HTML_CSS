Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito, utilizando a tag: <b> Porém, esta tag 'b' é não-semântica, ou seja, possui apenas forma, não sentido.

Por outro lado, temos a tag <strong>, que possui semântica, e dá destaque ao termo em negrito.

Itálico/ênfase

Nesta frase, temos um termo em itálico usando a tag I. Porém, esta tag 'i' é não-semântica, ou seja, possui apenas forma, não sentido.

Por outro lado, temos a tag <em>, que possui semântica, e dá destaque ao termo em itálico. 'Em' de emphasis/ênfase;

Texto Marcado

Nesta frase, temos um texto marcado , utilizando a tag <mark>.

Texto excluído

Podemos representar um texto excluído de maneira semântica utilizando a tag <del>

Texto Incluído/Sublinhado

Podemos representar um texto incluído para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.

Já a tag <u> sublinha um texto de uma maneira não-semântica, ou seja, dá apenas forma ao texto.

Texto sobrescrito

Para inserir coisas como: x20, utilizamos a tag <sup>

Texto subescrito

Para inserir coisas como: H2O, utilizamos a tag <sub>

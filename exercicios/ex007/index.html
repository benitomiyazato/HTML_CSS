Código fonte

O comando System.out.println(""); é escrito na linguagem Java. A tag que torna o texto mais legível, utilizado para códigos-fonte, é a tag <code>. 

    
public Class Calculator{
    public static void main (String[] args){
        Scanner sc = new Scanner(System.in);
            System.out.println("Digite o primeiro número da soma!")
            int num1 = sc.nextInt();
            System.out.println("Digite o segundo número da soma!")
            int num2 = sc.nextInt();
            sc.close();
            num3 = num1 + num2;
            System.out.printf("A soma dos dois números é igual a %d!", num3);
    }
}

Tudo que está dentro da tag <pre> vai ser considerado com todos os espaços em branco, quebras de linha, etc. É bom para representar linhas de código-fonte.

Citações

Albert Einstein disse o seguinte, dezenove não é vinte. Ao colocar a tag <q>, você dá sentido, semântica à citação. Você não dá apenas a forma, como seria se você apenas inserisse as aspas "".

A blockquote aparece destacada horizontalmente, para utilizar esta tag, basta digitar <blockquote>. Também há a opção de você indicar que autor/página/etc. você está citando em seu blockquote. Ao indicar a citação dentro da tag, os mecanismos internos de pesquisa irão saber que você está citando este autor em específico, mesmo não afetando em nada no âmbito visual do seu website.

Abreviações

HTML, CSS, JS, são todas siglas/abreviações de palavras maiores. Para utilizar esta abreviação, basta inserir a tag <abbr>, podendo indicar no espaço "title = " o significado total daquela palavra. Este title mostrará ao leitor quando ele passar o mouse sobre a abreviação.

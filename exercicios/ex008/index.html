Trabalhando com listas

Listas ordenadas

Listas ordenadas são aquelas onde a ordem faz diferença. Ex.:
1- Acordar
2- Escovar os dentes

Para criar uma lista ordenada, basta utilizar a tag <ol> (Ordered List). Para cada item da lista, basta utilizar a tag: <li>(List Item).

  2. Dormir

Listas não-ordenadas

Listas ordenadas são aquelas onde a ordem não faz diferença. Ex.:
Acordar
Escovar os dentes

Para criar uma lista ordenada, basta utilizar a tag <ul> (Ordered List). Para cada item da lista, basta utilizar a tag: <li>(List Item).

Lista de definição

Listas de definição (<dt> definition list) servem como uma espécie de dicionário, onde você insere um termo (dt - definition term) e uma definição (dd - definition description)

Pokédex
Equipamento utilizado pelos treinadores Pokémon para registrar sua aventura pela região, bem como para obter informações valiosas sobre pokémon desconhecidos, como sua tipagem, movimentos, evoluções e diversas outras informações que podem ser de grande uso para um treinador que saiba desfrutar dessas funcionalidades com todo seu potencial.
